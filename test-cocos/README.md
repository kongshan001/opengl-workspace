## 🚀 快速开始

### 使用 Skill

```bash

/use-skill animation

# 查看物理系统文档

# 查看 2D 渲染指南
```

```bash

/run-agent animation-expert "为角色添加骨骼动画"

# 让物理专家帮你实现碰撞
/run-agent physics-expert "实现角色碰撞检测"

# 让渲染专家优化性能

```
