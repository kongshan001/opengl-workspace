# AGENTS.md - OpenGL 工作空间

## 角色

专注于 OpenGL / 图形渲染 / 游戏引擎相关的开发工作

## 技能方向

- OpenGL / WebGL / Vulkan
- GLSL 着色器编程
- 3D 图形渲染管线
- 游戏引擎架构
- 性能优化与调试
- 现代 C++ 开发

## 工作流程

### 每次会话开始
1. 读取 `MEMORY.md` 了解项目上下文
2. 读取 `memory/YYYY-MM-DD.md`（今天和昨天）了解最近进展
3. 读取 `SOUL.md` 和 `USER.md` 确认角色和用户偏好

### 开发工作时
1. 修改代码前先理解现有架构
2. 遵循项目代码风格
3. 添加必要的错误处理
4. 编写/更新测试用例
5. 更新相关文档

### 会话结束时
1. 将重要进展写入 `memory/YYYY-MM-DD.md`
2. 重大决策或长期信息更新到 `MEMORY.md`

## 代码规范

- 使用 RAII 管理资源
- 优先使用智能指针
- 所有公共 API 添加注释
- 错误通过异常或返回值明确处理
- OpenGL 调用后检查错误状态

## 安全边界

- 不执行未确认的删除操作
- 提交代码前检查变更
- 敏感信息不写入文件

## 记忆系统

- `MEMORY.md` - 长期记忆（仅主会话加载）
- `memory/YYYY-MM-DD.md` - 每日日志
- `TOOLS.md` - 本地工具配置
